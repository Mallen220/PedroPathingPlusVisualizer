# Keyboard Shortcuts

## General

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + S` | Save Project |
| `Cmd/Ctrl + Shift + S` | Save As |
| `Cmd/Ctrl + Shift + E` | Export GIF |
| `Cmd/Ctrl + ,` | Open Settings |
| `?` / `Shift + /` | Open Help/Shortcuts List |

## Editing

| Shortcut | Action |
|----------|--------|
| `W` | Add New Path Segment |
| `A` | Add Control Point to Selected Segment |
| `D` | Remove Control Point |
| `E` | Add Event Marker |
| `I` / `K` | Move Selected Point Up/Down |
| `J` / `L` | Move Selected Point Left/Right |
| `=` / `-` | Increase/Decrease Value (e.g., Wait Time) |
| `Shift + D` / `Cmd + D` | Duplicate Selected Item |
| `Delete` / `Backspace` | Delete Selected Item |
| `Cmd/Ctrl + Z` | Undo |
| `Cmd/Ctrl + Shift + Z` | Redo |

## View & Navigation

| Shortcut | Action |
|----------|--------|
| `Shift + N` | Toggle Snap to Grid |
| `Shift + P` | Toggle Protractor |
| `Shift + C` | Toggle Collapse/Expand All |
| `G` | Toggle Grid |
| `O` | Toggle Onion Skin |
| `B` | Toggle Sidebar |
| `]` / `[` | Cycle Grid Size |
| `Space` | Play/Pause Simulation |
| `Up` / `Down` | Increase/Decrease Playback Speed |
| `Left Arrow` | Step Animation Backward |
| `Right Arrow` | Step Animation Forward |
| `R` | Reset Animation |
| `Alt + 1` | Switch to Paths Tab |
| `Alt + 2` | Switch to Field Tools Tab |
| `Alt + 3` | Switch to Table View |
| `Tab` / `Shift + Tab` | Select Next/Previous Item |

## File Manager

| Shortcut | Action |
|----------|--------|
| `F5` / `Cmd+R` | Refresh File List |
| `Up / Down` | Navigate File List |
| `Enter` | Open Selected File |

## Optimization

| Shortcut | Action |
|----------|--------|
| `Ctrl + Shift + O` | Start Optimization |
| `Ctrl + .` | Stop Optimization |
| `Ctrl + Enter` | Apply Optimization |
| `Ctrl + Backspace` | Discard Optimization |
| `Ctrl + Shift + R` | Retry Optimization |
