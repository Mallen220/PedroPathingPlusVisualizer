# Configuration & Settings

Access the settings menu by clicking the gear icon in the top navigation bar.

## Robot Configuration

These settings are critical for accurate collision detection and simulation.

| Setting | Description |
|---------|-------------|
| **Robot Width** | The width of your robot (left-to-right) in inches. |
| **Robot Length** | The length of your robot (front-to-back) in inches. |
| **Team Number** | Used for generating package names in code export. |

## Field Settings

- **Field Image**: Select the background image for the field (e.g., CenterStage, Freight Frenzy).
- **Custom Field Map**: You can upload a custom image if the standard ones don't fit your needs.

## Application Preferences

- **Theme**: Toggle between Light and Dark mode.
- **Default Path Color**: Choose the color for new path segments.
- **Project Directory**: The default folder where your `.pp` files are saved.

## Key Bindings

You can customize keyboard shortcuts in the "Shortcuts" editor within the Settings menu. See [Keyboard Shortcuts](Keyboard-Shortcuts) for the default list.
