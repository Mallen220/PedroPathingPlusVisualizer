### What's New!

## **Features:**

### **Configurable Telemetry**

- **Your Choice**: Select your preferred telemetry backend for generated code.
- **Options**: Choose between **FTC Dashboard**, **Panels**, **Standard** (Driver Station), or **None**.
- **Easy Switching**: Change this setting in **Settings > Auto Export** or via the **Quick Settings** in the Code tab.

### **Export & Workflow**

- **Direct Download**: Download generated code as a `.java` file directly from the Code tab using the toolbar or `Cmd/Ctrl + J`.
- **Quick Copy**: Use `Cmd/Ctrl + Shift + C` to copy generated code and `Cmd/Ctrl + Shift + T` to copy the waypoint table.
- **Smart Filenames**: The downloader automatically suggests a filename based on your project.

### **Live Code Preview**

- **Real-time Code Generation**: See your Java code update instantly as you modify your path.
- **Smart Diff View**: Additions are highlighted in green and removals in red, making it easy to track changes.
- **Enable It**: To start using this feature, go to **Settings > Auto Export** and toggle **"Auto Export Code"** to ON. A new "Code" tab will appear in the editor.

### **Follow Robot Mode**

- **Automatic Camera Tracking**: The field view now follows your robot during playback, keeping the action centered.
- **Seamless Control**: Manually panning or zooming temporarily pauses tracking. Tracking resumes automatically when you restart playback.
- **Enable It**: Toggle this feature in **Settings > Interface > Follow Robot**.

### **Accessibility & UI**

- **File Manager**: The toolbar is now fully keyboard-accessible with improved focus indicators.
- **Shortcut Hints**: Buttons in the Waypoints table now display their keyboard shortcuts in tooltips.
