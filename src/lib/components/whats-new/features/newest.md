### What's New!

## **Features:**

### **Sequential Command Export**

- **Command-Based Support**: You can now export your paths as a **Sequential Command** for use with libraries like **SolversLib** or **NextFTC** (experimental).
- **Flexible Integration**: Automatically generates the necessary boilerplate, including imports and command structures.
- **How to use**: Select **Sequential Command** in the Export Code dialog.

### **Embed Pose Data in Code**

- **Single-File Solution**: You can now choose to embed pose data directly into the generated Java file.
- **No External Files**: Removes the dependency on loading `.pp` files at runtime.
- **Enable It**: Check the **Embed Pose Data in Code** option in the Export Code dialog.

### **Search in Code Export**

- **Find Fast**: A new search bar in the Export Code dialog helps you quickly locate specific lines, variables, or coordinates in the generated code.
- **Keyboard Shortcut**: Press `Ctrl+F` (or `Cmd+F`) while the dialog is open to start searching.
