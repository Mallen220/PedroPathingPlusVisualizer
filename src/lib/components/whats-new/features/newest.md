### What's New!

## **Features:**
- **Adjustable Timeline Zoom:** Added zoom controls (In, Out, Reset) to the playback timeline. This makes it easier to place and edit event markers precisely, especially on longer paths. You can zoom using the new buttons or by holding `Ctrl` and scrolling over the timeline track.

## **UI Improvements:**
- **Improved Resizer Handles:** Enhanced the hover and focus styling for the layout resizer handles, making them more visible and easier to interact with via keyboard or mouse.
