### What's New!

## **Features:**

### **Enhanced Keyboard Navigation**
You can now navigate between sequence items using keyboard shortcuts:
* **Next Item:** Press `Tab` or `Ctrl` / `Cmd` + `Down Arrow`
* **Previous Item:** Press `Shift` + `Tab` or `Ctrl` / `Cmd` + `Up Arrow`

## **Bug Fixes:**

### **Code Export Improvements**
* Fixed an issue where tangential start headings were not correctly handled when exporting code. The starting heading is now calculated correctly based on the path.
