# What's New in 2.2.0!

Improved selection and workflow, and event marker logic for multi-platform command support.

## **Visualizations & Field Rendering**

- **Live Timeline Hover Scrubbing:** Added a Ghost Robot preview when hovering over the timeline.
- **Customizable Robot Features:** You can now attach and customize physical robot features directly on the robot image. To add a new feature, go to the **Robot** tab in **Settings**.
- **Velocity Tooltips:** Added a new setting (off by default) that shows the Velocity, Time, and distance traveled at any point in a path. This is in the interface section of Settings under "Velocity Tooltips".
- **New Robot Image:** Added a brand new turtle robot image option. Check it out :)

## **Workflow Enhancements & Shortcuts**

- **Shift+Drag Box Selection:** Quickly select multiple waypoints at once using Shift+Drag Box Selection in the FieldRenderer.
- **Lock Field View Shortcuts:** Added new shortcuts to Lock the Field View (alt+g) and to Toggle the Field View Lock (alt+l).
- **Select All Keybinding:** Added a quick shortcut (ctrl+a) to select all elements in the current path.
- **Java Code Importer Improvements:** Improved parsing and support for imported Java event markers.
- **Discord Sidebar Link:** Replaced the GitHub link in the sidebar with a Discord link. The GitHub link has been moved to the settings header, and can easily be re-added to the sidebar.

## **Architecture & Core Logic**

- **Event Marker Overhaul:** Overhauled event marker logic to support the FTC SDK, SolversLib, and NextFTC (Runnables only). **Note: Users must update to the newest TurtleTracerLib version for proper support.**
- **Dynamic Library Notifications:** Added a new Toast notification that alerts you when `TurtleTracerLib` updates are available.
- **Improved MS Store Update Notifications:** Refined the Microsoft Store update notification for a better user experience.
- **Code Cleanup:** Performed general code cleanup and maintenance across the application.
