# Robot Profiles

**Robot Profiles** allow you to save and switch between multiple robot configurations. This is especially useful if you manage multiple robots (e.g. competition bot vs. test bot) or want to test different physics tuning constants.

---

## ‚ûï How to Add a Profile

1.  Open **Settings** (click the gear icon or press `Cmd/Ctrl + ,`).
2.  Expand the **Robot Configuration** section.
3.  Click the **+ New Profile** button.
4.  Enter a name for your profile (e.g., "Competition Bot").
5.  Click **Save Current Settings**. This will create a new profile using whatever settings are currently active.

## üõ†Ô∏è Designing & Editing Profiles

To modify an existing profile:

1.  **Load the Profile:** Select the profile from the dropdown and click **Load**. This applies the profile's settings to the visualizer.
2.  **Make Changes:** Adjust any robot settings (Dimensions, Velocity/Acceleration limits, Friction, etc.) in the Settings menu.
3.  **Update the Profile:**
    - Ensure the correct profile is selected in the dropdown.
    - Click the **Update** button (save icon) next to the delete button.
    - Confirm the update.

> **Tip:** You can also create a *new* profile based on your current adjustments by clicking "+ New Profile" instead of updating the existing one.

---

## üöÄ Using Profiles

To switch robots:

1.  Open **Settings** > **Robot Configuration**.
2.  Select your desired profile from the dropdown menu.
3.  Click **Load**.

The visualizer will immediately apply the saved physics constants and dimensions.

If you no longer need a profile:

3.  Confirm the deletion.
